## Mid-Term Workshop: AI-Aided Translation

### 1. Objective
- 바이브 코딩을 활용하여 local llm을 번역에 사용할 수 있다.
- OCR을 활용하여 이미지 내 텍스트를 추출할 수 있다.
- AI를 활용한 번역 workflow를 적용할 수 있다.
- 다양한 metric을 활용하여 번역 품질을 평가할 수 있다.

### 2. Scope & Content
- **Topics:** open models (llm, ocr, translation-specified llm, ...), evaluation metrics (from BLEU to model-based ones)
- **Format:** Google Colab, llm chat-bot을 통한 코드 작성

### 3. Preparation & Workflow
1. Data Collection: 1000개 이상의 영어 문장이 들어있는 image files (드래그 허용x)
2. Notebook Setup: 수강생들이 이용할 수 있는 ipynb 파일
3. validaiton script (수강생들이 제출한 결과물을 평가할 파이썬 스크립트)

### 4. Tech Stack
- Python, Hugging Face, Git, etc...
