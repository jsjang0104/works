# Poor Man’s Quality Estimation: Predicting Reference-Based MT Metrics Without the Reference

----

**Author:** Vilem Zouhar et al.

**Journal/Year:** EACL 2023

https://arxiv.org/pdf/2301.09008

----

## Introduction
- Data augmentation and synthetic QE data via automated metrics has already been explored, **though never in the pre-training & fine-tuning fashion**

### 연구 흐름 및 결과
- step1: src + ref 코퍼스를 BLEU, TER 등의 reference-based metric을 이용하여 label 생성
- step 2: step1에서 만든 metric 점수를 예측하는 모델(BiLSTM)을 학습시킴 (pre-training)
  - 결과: sentBLEU 기준 pearson correlation 60.4%
- step 3: step2에서 학습시킨 모델을 소량의 human annotation data를 이용하여 fine-tuning
  - 결과: from scratch로 학습할 때 20%였던 pearson 상관계수가 TER로 학습 후 파인튜닝할 때 23%로 성능이 향상됨

### Research Question 
- RQ1: reference없이 reference-based metric 결과를 추론할 수 있는가?
  - 그렇다. sentBLEU 60.4% 달성.
- RQ2: reference-based metrics의 추론에 data size가 주는 영향은?
  - 질 좋은 source + reference corpus가 필요함. 만약 데이터 부족시 bean search를 이용하여 여러개의 번역 결과물을 뽑을 수도 있음.
- RQ3: 다른 MT 시스템 간 transfer가 성능에 얼마나 영향을 주는가?
  - 하나의 번역 모델 결과로만 학습해도, 다른 번역 모델 결과에 적용할 때 성능 저하가 아주 약간에 불과함.
- RQ4: ME가 QE를 위한 pre-training step이 될 수 있는가?
  - 그렇다. TER pretraining 후 fine-tuning했을 때 성능 향상
 
### Experiment Setup
- Data
  - WMT의 English --> German sentences로 ME 구축

## 들었던 생각 
### 내 연구와의 차이점 
- 이 논문의 QE 모델: BiLSTM  
  - 내가 연구에서 쓰고 싶은 것: 다목적 LLM을 fine-tuning
- 이 논문의 학습 순서: ME(src+ref)로 pre-training, human assessment(src+DA)로 fine-tuning
  - 내가 연구에서 하고 싶은 것: silver + gold 또는 silver만으로 fine-tuning하여 gold로 성능 검증
- 이 논문이 다룬 언어쌍: EN->DE로 아주아주 고자원 언어
  - 내 연구에서 다룰 예정: EN->인도소수어들 (아주아주 저자원 언어)
  - 혹은 내 논문에서 인도 언어 데이터가 분량 때문에 한계가 생긴다면 이 논문을 참고하여 EN->DE로 방향을 변경해볼 수도 있을 것 같음. (EN->인도소수어들은 example 단위의 analysis가 불가능하지만 이 경우는 가능)
- 공통점: source + reference corpus로 구축하는 자동 메트릭 점수 (= ME = silver label)
### 기타
- src+ref corpus에서 데이터 부족시 bean search를 통해 데이터를 증가하는 방법을 제시해줌. 내 연구에서 동일한 상황 발생 시 고려 가능
