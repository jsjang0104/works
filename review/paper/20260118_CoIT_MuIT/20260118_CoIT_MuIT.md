# Extrapolating Large Language Models to Non-English by Aligning Languages

- https://arxiv.org/abs/2308.04948

## 요약

- CoIT, MuIT를 통한 LLMs의 non-English ability 향상
- cross-lingual instuction tuning에 대한 scaling law, data allocation strategy 정립 (리소스 제한이 있는 상황 대비)
- cross-lingual instruction-tuning / continued pre-trainineg의 scaling laws을 비교 -> aliging language is more efficient

### 배경
- 현존 LLM에서는 영어와 타 언어 사용 간 능력치가 차이가 많이 남 (due to the imbalance in training data)
  - 타 언어가 구조적으로 영어랑 차이가 많이 날 수록 성능도 더 떨어짐

### 핵심 아이디어
- pre-trained LLMs의 non-English에서의 ability 향상: by building semantic alignment cross English and non-English languages.
  - [1 단계] 개별 언어 타겟팅: Cross-Lingual Instruction Tuning (CoIT) on LLaMa -> cross-lingual model 획득 (x-LLaMAs, 특정 언어 쌍 (예를 들면 영어-한국어)에 특화)
  - [2 단계] Scaling Law 분석 by Multilingual instruction-tuning (MulT) -> scalable translation data 조사 (번역 데이터와 모델 성능이 어느 정도의 비례 관계를 갖고 있는가?)
    - 개별 언어에 대한 분석을 통합: mixed sources from previous stages -> mulitilingual m-LLaMA
    - Practical Scenario: instruction-tunings을 특정 데이터 예산 내에서 한다고 가정

- optimization: optimal data allocation
  - 이전 연구에서의 scaling laws를 활용하여 task를 non-linear promgramming으로 함
  - 이러한 최적화의 목적: 모든 언어의 평균 점수를 최대한으로 끌어올림


### 실험 세팅
- pre-trained LLM: LLaMA-7B
- target languages: 아랍어, 그리스어, 힌디어, 터키어, 베트남어, 중국어 (영어와 적은 Alphabet을 공유)
- Instruction Tuning Details: Parallel corpora 사용 
- Benchmarks: XQUAD, MLQA, FLORES-101
- Evaluation Metrics: ChatGPT가 평가, excact-matching results 등


### 결과

- XQUAD, MLQA에서 x-LLaMAs는 6개 비영어권 언어에서 영어 기반 모델 (Alpaca) 보다 27.83% 성능 향상
- FLORES-101에서 x-LLaMAs는 이전 baselline 라마 모델 보다 18.89% 성능 향상

- 결론
  - m-LLaMA는 개별 언어들에서 x-LLaMAs만큼의 성능을 보임
  - m-LLaMA는 multilingual instruction을 따를 수 있음

- Response content & Representation space에 대한 추가 분석 -> multilingual semantic space가 m-LLaMA의 중간 layer에 aligned

- 소스가 제한적인 세팅(1.2M)에서는 optimized data allocation이 기존보다 더 좋은 성능을 이끌어냈음


## 의견
- 언어 고르는 기준이 "영어와 공유하는 알파벳이 적은 것"으로만 했다고 하는데, 너무 naive한 기준인 것 같다. 
- 중국어는 알파벳도 물론 없고 어순 등 문법구조나 발달한 어족 자체가 영어와 완전 딴판인 언어이다. 
- 베트남어는 한자 문화권인데 문자만 알파벳이다. 즉 영어와 공유하는 알파벳은 많지만 어휘는 딴판이다.
- (내가 아는 한) 터키어는 독일어랑 비슷한데, 독일어는 영어랑 완전 형제 관계인 언어이다. (어순, 어휘 등 매우 유사)
- 이러한 차이점이 디테일하게 반영됐었으면 좋을듯?

- Multilingual LLM을 다룰 때 CoIT, MuIT의 방법론은 기억해두기
